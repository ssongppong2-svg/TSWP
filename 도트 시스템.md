# 목적 (Purpose)

본 문서는 TSWP의 모든 픽셀아트 제작 규칙을 정의한다.

모든 캐릭터, 보스, 몬스터, 오브젝트, UI는
동일한 도트 규칙을 따라 제작한다.

---

# 방향

모든 캐릭터는

좌우 반전(Flip)

방식을 사용한다.

좌측 전용 스프라이트는 제작하지 않는다.

---

# 색상

팔레트는

공통 팔레트를 사용한다.

색상 수는

32~64색을 권장한다.

---

# 그림자

모든 캐릭터는

바닥 그림자를 가진다.

그림자는

원형

반투명

고정 크기를 사용한다.

---

# 광원

실시간 광원을 사용하지 않는다.

필요한 경우

Sprite 기반 광원만 사용한다.

---

# 카메라

# UI

# 제작 규칙

□ 외곽선 통일

□ 팔레트 공유

□ 그림자 통일

□ 애니메이션 속도 통일

□ 캐릭터 비율 통일

□ Pixel Perfect 유지

---

# AI 제작 규칙

Claude/Fable AI를 사용할 경우

반드시 아래 규칙을 따른다.

• 정면이 아닌 측면(2D Side View)

• 투명 배경

• Pixel Art

• 32x32 비율 기준

• Sprite Sheet 출력

• 같은 팔레트 유지

• 캐릭터 중심 정렬

---

# 개발 원칙 연결

• 모든 캐릭터는 동일한 도트 규칙을 따른다.

• 모든 보스는 같은 스타일을 유지한다.

• 모든 UI는 픽셀 스타일을 유지한다.

---

# 테스트 체크리스트

□ 모든 캐릭터 크기가 동일한가

□ Pixel Perfect가 유지되는가

□ 애니메이션이 자연스러운가

□ 팔레트가 통일되어 있는가

□ 확대해도 깨지지 않는가

□ AI가 같은 스타일을 유지하는가
